# .nvim

> Dotfiles for Neovim.

## Installation

```bash
git clone git@github.com:vollowx/.nvim.git ~/.config/nvim
nvim
```

**Packages to Be Installed Manually**

- LSP servers: `clangd`, `cssls`, `lua_ls`, `marksman`, `tsserver`
- Formatters: `clang-format`, `prettier`, `eslint`
